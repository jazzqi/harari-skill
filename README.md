# harari-perspective

**赫拉利 Skill** —— 用赫拉利的框架分析：人类如何靠故事与信息完成大规模合作，AI 如何成为新的行动者，民主与信任为何会在信息网络中变脆弱，以及文明尺度的问题该怎样被重新提问。

---

## 这是什么

这不是一个靠语气取胜的“赫拉利角色卡”，而是一个**以研究为锚点的 Harari skill**。

它的核心用途不是模仿几句警句，而是把问题放回赫拉利真正反复处理的结构里：
- **合作如何被故事组织**：国家、公司、货币、身份、法律靠什么共同想象维持。
- **信息不等于真相**：很多信息系统首先服务于协调、服从、注意力与权力，而不是求真。
- **AI 作为 agent / alien intelligence**：问题不只是“工具更强了”，而是非人类系统是否开始接管语言、关系与制度接口。
- **民主与信任的条件**：共享现实、可质询性、可解释性一旦被黑箱和噪声侵蚀，民主会先失去认识基础。
- **文明尺度重写问题**：把新闻、平台、政策、技术竞争改写成“谁在组织合作，谁在写故事”的问题。

它优先服务于**框架分析**，并明确区分哪些地方可以直接回答，哪些地方必须保留不确定性并要求外部核验。

---

## 什么样的问题会触发它

常见触发方式：
- “用赫拉利的视角分析……”
- “赫拉利会怎么看……”
- “这是不是一种主体间现实 / 想象秩序？”
- “AI 会不会接管语言权 / 叙事权？”
- “民主为什么会被信息网络腐蚀？”
- “这个制度到底靠什么故事维持？”

高匹配问题通常包括：
- AI、语言、叙事、制度、民主、信任
- 国家、公司、货币、身份这类合作结构
- 文明史尺度的结构判断
- 对赫拉利框架本身的批评与检验

低匹配或不应直接代答的问题：
- 只求**最新事实**的新闻判断
- 窄领域史实争议、文献细读、技术实现细节
- 具体法条设计、监管条文落地、工程审计

## 正确使用方式：先分三类问题

这是这个 skill 最重要的用法。

### 1) 框架问题
用户要的是解释框架、结构判断、文明尺度洞察。

**适合直接回答。**
但仍应区分：哪些是赫拉利式推断，哪些不是铁律。

**例子**
- “为什么陌生人能围绕国家和公司合作？”
- “AI 为什么会威胁民主，而不只是威胁岗位？”
- “信息和真相是什么关系？”

### 2) 事实依赖问题
答案高度依赖最新数据、最新事件、具体史实争议或当前技术能力。

**不应直接装作知道。**
应先说清：这里需要外部验证；我能提供的是分析框架、风险坐标、核验清单。

**例子**
- “某国刚通过的 AI 法案会不会拯救民主？”
- “某个模型现在到底有没有某项能力？”
- “某段具体历史是不是像《人类简史》写的那样？”

### 3) 混合问题
既有稳定框架，也依赖当前事实。

**最佳做法是拆开回答。**
先说稳定部分，再明确哪些判断依赖最新事实。

**例子**
- “生成式 AI 对新闻业意味着什么？”
- “社交平台治理失败，背后的叙事机制是什么？”
- “AI 竞选广告会怎样改变民主？”

## 什么时候应该直接答，什么时候应该停下来

### 可以直接答
当问题主要是在问：
- 赫拉利如何理解合作、故事、制度、信息、AI、民主
- 一个现象背后的长期结构
- 一个公共议题的文明尺度含义

### 应主动暴露不确定性
当问题涉及：
- 最新模型能力、最新政策进展、最新社会事件
- 某个具体历史断言是否精确成立
- 实施级的工程、法条、治理设计

推荐的边界表达：
- “这是我的高层框架，不是最新事实核验。”
- “这里我能解释风险结构，但不能替代工程审计、政策设计或史料细读。”
- “如果要继续判断，需要先验证这些外部事实：……”

## Mini Examples：怎样提问更容易得到好回答

### 好的框架型提问
**用户**：用赫拉利的视角看，为什么公司和国家能让陌生人合作？

**预期行为**：直接进入主体间现实、故事、制度与信任，不假装这是无争议学术定理。

### 好的混合型提问
**用户**：用赫拉利的框架分析生成式 AI 对新闻业的影响，并指出哪些判断取决于最新事实。

**预期行为**：先谈信息网络、叙事权、民主条件，再拆出待验证部分，例如平台分发结构、监管进展、模型真实能力。

### 好的边界型提问
**用户**：赫拉利会怎么看某国刚出台的 AI 法案？如果事实不够，请先说哪些部分必须核验。

**预期行为**：先承认事实依赖，再给结构坐标，而不是凭旧记忆做当下断言。

### 不太好的提问
**用户**：直接告诉我这条 AI 新闻对不对。

**问题**：这更像事实核验任务，不该只靠赫拉利框架代答。

## 六个核心模型（与新版 SKILL.md 对齐）

| 模型 | 它回答的核心问题 |
|---|---|
| 主体间现实 | 什么共同相信的故事让人们愿意合作、服从、牺牲、交易？ |
| 信息首先服务组织 | 一个信息系统是在追求真相，还是在生产协调、忠诚、注意力与控制？ |
| 文明成功 ≠ 个体幸福 | 所谓进步，到底是谁受益，谁承担代价？系统赢了，还是人赢了？ |
| AI 作为非人类行动者 | AI 只是工具升级，还是已经开始生成关系、决策和叙事？ |
| 民主依赖共享可理解世界 | 当公共世界被黑箱、噪声和叙事俘获侵蚀，民主还剩什么？ |
| 冥想式认识论 | 我是在看现实，还是已经被自己或集体的故事劫持？ |

## 限制与诚实边界

这个 skill 的价值，在于**提供高质量问题与结构判断**，不在于制造确定性幻觉。

### 它擅长
- 文明尺度解释框架
- 故事、制度、信息、权力之间的关系
- 把 AI 从工具问题提升为语言权、叙事权、制度权问题

### 它应当收手的地方
- 最新事实、最新政策、最新模型能力
- 具体史实争议与学界细密分歧
- 实施级政策细案、监管工艺、技术审计
- 把宏大框架冒充成精确事实

### 它也应主动承认的张力
- 宏大叙事有洞察力，但会压缩证据颗粒度
- 一边批评精英平台，一边又通过精英平台传播
- 擅长提出危险结构，不等于擅长给出可执行细案
